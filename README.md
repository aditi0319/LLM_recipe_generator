# LLM_recipe_generator
# 🍳 AI Recipe Generator

An AI-powered recipe generator built using Streamlit and Google Gemini API.

## Features

- Generate recipes from ingredients
- Save recipes
- Search saved recipes
- Download recipe as TXT
- Responsive Streamlit UI

## Tech Stack

- Python
- Streamlit
- Google Gemini API
- JSON

## Run Locally

```bash
pip install -r requirements.txt

streamlit run app.py
```
